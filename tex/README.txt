
===============================================================================
                                File Descriptions
===============================================================================

resume-format.cls
--------------
- This is the main class file that defines the structure and layout of the resume.
- Include this in your LaTeX document with \documentclass{resume-format}.

fontawesome.sty
---------------
- This style file includes the FontAwesome icons used in the resume.
- Automatically loaded by the class file.

fonts/
------
- This directory contains all the fonts required by the resume template.
- Includes FontAwesome and various weights of the Roboto and SourceSansPro fonts.

README.txt
----------
- This file you are currently reading.
- Provides an overview of the directory structure and the purpose of each file.

resume.aux, resume.log, resume.out
----------------------------------
- Auxiliary files generated by LaTeX during the compilation process.
- These files can be ignored and are automatically created when compiling resume.tex.

resume.tex
----------
- Sample LaTeX document that uses the resume-format class.
- Customize this file to create your own resume.

===============================================================================
                               Usage Instructions
===============================================================================
1. Ensure that you have LaTeX and xelatex installed on your system
2. Clone the contents of this repository, and navigate into it
3. Run `xelatex resume.tex` to generate the PDF, or use Overlea (https://overleaf.com/)
4. From the parent directory, you can also run `make` to generate the PDF with the content of resume.yml
